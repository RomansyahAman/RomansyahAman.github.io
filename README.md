# Personal Portfolio - Muh. Aman Romansyah

Welcome to my personal portfolio! This project is a minimalist, retro-tech themed dashboard designed to showcase my skills and projects in IoT, Robotics, and Programming.

## Features
- **Retro-Tech Terminal Aesthetic**: Dark mode design with a simulated command-line interface.
- **Responsive Layout**: Optimized for Desktop, Tablet, and Mobile.
- **Interactive Elements**: Typing animations and project repository grid.

## Tech Stack
- HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JavaScript
